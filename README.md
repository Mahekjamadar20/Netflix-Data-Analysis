# Netflix Data Analysis — EDA Project

## About This Project
Exploratory Data Analysis on Netflix dataset of 8,800+ titles 
across 190+ countries using Python.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Dataset
- Source — Kaggle
- Rows — 8,800+ titles
- Columns — 12

## Key Findings
- Netflix shifted 70% toward TV Shows after 2015
- USA is #1 content producer, India is #2
- TV-MA is most common rating — adult audience
- Top genres — International Movies and Dramas
- Content grew rapidly 2016 to 2019
- January and July have most content additions

## How To Run
1. Download the notebook
2. Install — pip install pandas numpy matplotlib seaborn
3. Open in Jupyter Notebook
4. Run all cells
